Introducción

Un router TP-Link permite compartir una conexión a Internet mediante cable o red inalámbrica (Wi-Fi). Para configurarlo correctamente se debe ingresar a su panel de administración desde un navegador web.

Paso 1. Conectar el Router

  • Conecte el cable del proveedor de Internet al puerto WAN.
  • Conecte la computadora mediante un cable Ethernet o Wi-Fi.
  • Encienda el router.

Paso 2. Acceder al Panel

Abra el navegador y escriba una de las siguientes direcciones:

  • http://192.168.0.1
  • http://192.168.1.1

Ingrese el usuario y contraseña del router. En muchos modelos el usuario y contraseña inicial son:

  • Usuario: admin
  • Contraseña: admin

Paso 3. Configurar Internet

Seleccione el tipo de conexión proporcionado por su proveedor:

  • IP Dinámica (DHCP)
  • PPPoE
  • IP Estática

Paso 4. Configurar la Red Wi-Fi

  • Escriba el nombre de la red (SSID).
  • Seleccione seguridad WPA2 o WPA3.
  • Ingrese una contraseña segura.
  • Guarde los cambios.

Paso 5. Reiniciar

Después de guardar la configuración, reinicie el router para aplicar los cambios.

Recomendaciones de Seguridad

  • Cambiar la contraseña del administrador.
  • Actualizar el firmware del router.
  • No compartir la contraseña del Wi-Fi.
  • Utilizar una contraseña fuerte con letras, números y símbolos.
  • Desactivar WPS si no se utiliza.
